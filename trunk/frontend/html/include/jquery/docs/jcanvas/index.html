jCanvas

jQuery meets the HTML5 canvas

jCanvas is a jQuery plugin that makes
the HTML5 canvas easy to work with.

Download jCanvas Fork Me!

Features

  • A flexible API for drawing basic and complex shapes
  • Works on desktop and mobile
  • Small size – 23.6kb minified
  • Free and open-source

Syntax

jCanvas's simple syntax makes drawing shapes as easy as it should be.

// Draw a circle
$("canvas").drawArc({
  fillStyle: "green",
  x: 100, y: 100,
  radius: 50
});

For animation, drag-and-drop, vectors, and much more, check out the Documentation.
