Ellipses

You can draw an ellipse using the same basic properties used to draw a rectangle.

$("canvas").drawEllipse({
  fillStyle: "#c33",
  x: 150, y: 100,
  width: 200, height: 100
});

$("canvas").drawEllipse({
  strokeStyle: "#36c",
  strokeWidth: 4,
  x: 150, y: 100,
  width: 200, height: 100
});
