jCanvas (minified) jCanvas (source)

jCanvas requires jQuery (version 1.4 or newer). If you're already using jQuery, you can just add jCanvas to your page (via <script>).

jCanvas is licensed under the MIT license, which you can view on GitHub.

Supported Browsers

  • Google Chrome 10+
  • Mozilla Firefox 6.0+
  • Apple Safari 4.0+
  • Opera 10.1+
  • Internet Explorer 9+

For Internet Explorer 8 and older, try using jCanvas with FlashCanvas.
